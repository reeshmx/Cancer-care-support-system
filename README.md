The Cancer Care Support System is a role-based web application developed to support cancer care management by enabling effective interaction between administrators, doctors, and patients.
The system centralizes patient information, doctor coordination, and administrative control into a single, easy-to-use platform.

This project is built using PHP, HTML, CSS, and MySQL, and runs on a XAMPP local server environment.

**Objectives**

Provide a centralized system for cancer care support

Enable role-based access for Admin, Doctor, and Patient

Improve organization of patient records and appointments

Demonstrate full-stack web development using PHP & MySQL
